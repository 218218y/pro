## Stage 9 completed

### 1. Fixed a real overlay/export family contract regression

`npm run verify:closeout:write` exposed a contract failure in the overlay/export family:

```text
overlay quick actions dock should match /createQuickActionsDockController\(\{ api, reportNonFatal: reportOverlayAppNonFatal \}\)/
```

Changed file:

```text
esm/native/ui/react/overlay_quick_actions_dock.tsx
```

What changed:

- Restored `QuickActionsDock` to pass the shared reporter directly into `createQuickActionsDockController`.
- Removed the inline closure `(op, err) => reportOverlayAppNonFatal(app, op, err)` from the controller wiring.
- Removed `app` from the controller memo dependency list.
- Preserved App-scoped reporting in the local DOM event add/remove failure paths where `App` is already naturally available.

Decision:

- The controller is a low-level wiring/controller seam and must stay thin.
- Passing an inline App-bound closure through this controller was an overreach from Stage 7, even though it was well-intentioned.
- The professional fix is to obey the existing family contract instead of weakening the contract or adding another wrapper.
